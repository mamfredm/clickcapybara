What it actually does

Your search terms, intelligently classified by AI.

Most Google Ads accounts are leaking budget on search terms that will never convert — job seekers, competitors, irrelevant queries. Manually reviewing them is slow, error-prone, and doesn't scale. AI Max NegativeIQ automates the entire process.

Every day, the Fetcher pulls your latest search terms from Search campaigns into a Google Sheet. The Analyzer sends them in batches to an AI — which classifies each one based on your industry, your business, and your conversion data. Not generic rules. Actual context.

Every term gets a classification, a confidence score, and a one-line explanation of why the AI made that call. You review, correct if needed, and push negatives back into your account in one click. The system learns from your corrections over time.

Conversion protection — always on

Any search term that has driven a conversion is automatically protected. The tool will never flag it as a negative, regardless of how it looks on the surface.

What you get

Three components. One complete system.

The Fetcher

Google Ads → Scripts

Runs daily on a schedule. Pulls every search term from your Search campaigns, deduplicates them, and aggregates clicks and conversions over a 30-day window. Writes everything into your Google Sheet automatically — no manual export needed.

The Analyzer

Google Sheets → Apps Script

The intelligence layer. Sends your search terms to AI in scalable batches for fast, cost-efficient classification. Each term returns with a classification, a confidence score between 0 and 1, and a plain-language explanation. Fully config-driven: define your industry, categories, and a few examples once — the AI handles the rest.

The Google Sheet Template

Your dashboard

Four tabs, all automated. SearchTerms is your working surface — color-coded by classification, confidence-scored, with a one-click export. Config is the only thing you touch per account — industry, categories, example terms. Negative_Export holds validated negatives ready to push. Expansion_Ideas captures search terms worth building new campaigns around.

Setup guide

Up and running in minutes.

The Config and API key are a one-time effort. If you can paste text and click a button, you can run this.

1

Copy the Google Sheet template

Open the template from GitHub, make your own copy, and fill in the Config tab — your industry, business name, and 5–8 example search term classifications. This is the only customization you'll ever need to do.

2

Add the Fetcher to Google Ads

In Google Ads: Tools → Scripts → New. Paste the Fetcher code, replace the Sheet URL placeholder with your own Sheet URL, authorize it, and set it to run daily. That's it — it will run automatically every day from here on.

3

Add the Analyzer to your Sheet

In your Google Sheet: Extensions → Apps Script. Paste the Analyzer code, save, and authorize. A new menu item "AI Max NegativeIQ" will appear in your Sheet toolbar.

4

Run the setup menu

Click AI Max NegativeIQ → Prepare Layout. This styles all four tabs, adds color coding, and sets up the dropdowns. Run this once after setup, and again after each classification batch.

5

Let it run — then review

Next day, open the Sheet. Click Step 2 to classify all new search terms. Review the AI's suggestions, correct anything you disagree with, tick the checkbox to validate, and push negatives to your account in one click.

The full setup guide with screenshots is in the README on GitHub.

Under the hood

It's real code. Here's a slice of it.

fetcher_v1.js — AI Max NegativeIQ

// Pull search terms from Google Ads → write to Sheet

function fetchSearchTerms() {

// Only pull terms with real traffic — skip noise

const query = `SELECT

search_term_view.search_term,

campaign.name,

metrics.clicks,

metrics.conversions

FROM search_term_view

WHERE segments.date DURING LAST_30_DAYS

AND metrics.clicks > 0

`;

// Deduplicate — aggregate clicks across campaigns

const seen = {};

AdsApp.search(query).forEach(row => {

const term = row.searchTermView.searchTerm;

if (seen[term]) {

seen[term].clicks += row.metrics.clicks;

seen[term].conversions += row.metrics.conversions;

} else {

seen[term] = { clicks: row.metrics.clicks,

conversions: row.metrics.conversions };

}

});

// Write new terms to Sheet — skip already classified

writeNewTermsToSheet(seen, sheet);

}
